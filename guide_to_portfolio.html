Current Samples

Here you'll find samples of technical documentation I've created for those employers who published documentation externally. The documents are all PDFs, though some were originally published in HTML.

Employers and Sample Types

Intel Corp. Samples

Developer documentation created for Intel Parallel/Cluster/Embedded Studio XE command line tools and APIs. All are single-sourced XML, including DITA topic-based help and a tutorial.

  • Cookbook-style or explanatory topics

  • JIT API

  • 2 versions of Getting Started Tutorials on Static Analysis

Kavi Corp. Samples

Administrator and end-user documentation for Kavi Workspace, a collaborative platform and tool suite for standards organizations. Since the company supported open source as much as possible, online help was authored in Docbook XML via XEmacs on Linux, and published online as HTML and massive PDF books. Other types of documents were created in OpenOffice. Most graphics were created in Paint on a Windows OS test box, or with GIMP on Linux, or by other team members with OmniGraffle on Mac OS.

  • Help on various subjects including email, roles and permissions, billing

  • Release Notes

  • Product Summaries
